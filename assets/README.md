# Assets Folder

This folder contains images for the trading bot documentation.

## Required Images:

### 1. `logo-banner.png`
- **Size**: 800x200px (recommended)
- **Content**: Professional banner with "AI-Enhanced CoW Trading Bot" title
- **Include**: Base network logo, CoW Protocol logo, AI/robot icon
- **Style**: Clean, modern, tech-focused

### 2. `bot-running-screenshot.png`
- **Size**: 700x400px (recommended)  
- **Content**: Terminal/console showing the bot running
- **Show**: Wallet balance, trading decisions, price monitoring
- **Capture**: The actual bot output when it's running

### 3. `architecture-diagram.png`
- **Size**: 800x500px (recommended)
- **Content**: Flow diagram showing:
  - Wallet → Price Feeds → AI Analysis → CoW Protocol → Trades
  - Safety mechanisms and decision points
  - Data flow between components
- **Tools**: Use draw.io, Lucidchart, or similar

### 4. `telegram-notifications.png`
- **Size**: 600x400px (recommended)
- **Content**: Screenshot of Telegram chat with bot notifications
- **Show**: Order fills, profit updates, status messages
- **Privacy**: Blur any sensitive wallet addresses/amounts if needed

## Tips:
- Use consistent color scheme (blues/greens for trading theme)
- Keep text readable and professional
- Optimize images for web (PNG format, reasonable file size)
- Test how they look on both light/dark GitHub themes
